# AINotesCleaner
AI-powered tool that instantly cleans, organizes, and summarizes messy notes into structured study-ready content.

This app transforms messy, unstructured notes into clean, organized, and readable formats using AI.
Built for students who want clear notes without spending hours rewriting them.

Features:

- Cleans messy notes

- Adds headings and bullet points

- Generates summaries

- Highlights key terms

- Beginner-friendly and fast
